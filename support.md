# Codafy Support

Thank you for using Codafy.

If you need help with the application, have questions, want to report a bug, or would like to request a feature, please contact:

codafy.support@bryantord.com

## Before Contacting Support

Please include:

- Device model
- iOS version
- Codafy version
- Plex Server version
- Description of the issue
- Steps to reproduce the problem

## Response Time

We typically respond within a few business days.

## Common Topics

- Plex connection issues
- Playback issues
- Downloads
- Apple Watch synchronization
- Subscription questions
- Feature requests
